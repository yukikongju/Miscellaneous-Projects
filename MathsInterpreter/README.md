# Maths Interpreter

Expression: `sin(-3 * pi / 2) - tan(pi / 4) + 5`
Inorder traversal


- [ ] Lexer // Tokenizer: breakup the expression into tokens
- [ ] Parser: Construct AST following PEMDAS from tokens
- [ ] Evaluator: Takes AST and evaluate expression recursively
