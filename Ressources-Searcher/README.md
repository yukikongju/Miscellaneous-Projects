# Ressources Searcher

Search for course lectures, lecture notes, youtube videos, pdf and 
textbook of a subject you would like to learn

## Usage

Generate List of Ressources: ```res-search 'Subject'```

Flags:

- Y: youtube playlist
- T: textbook (and solutions)
- C: Coursera
- N: Lecture Notes
- P: problem set
- S: Solutions
