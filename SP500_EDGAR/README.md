# Edgar S&P 500

Fetching S&P 500 financial data from [SEC website](https://www.sec.gov/edgar/searchedgar/companysearch.html) using python edgar API

Todos:
- [ ] Fetch yearly/quarterly data for SP500 company
    - [ ] Income Statement
    - [ ] Balance Sheet
    - [ ] Cashflow
